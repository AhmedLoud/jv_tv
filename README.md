# Commandes
Z => Avancer
S => Reculer
Q => Strafer à gauche
D => Strafer à droite

A => Sprinter (appuyer une seule fois)
R => Recharger

P => Debloque la souris de l'ecran de jeu
Entree => Passer les fenêtres de dialogue / Menu
Echap => Quitter le jeu
